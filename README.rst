
django-custom-accounts-plus

This package stores password histories and checks new password with the history


Quick start
-----------
1. Add "password_histories" to your INSTALLED_APPS setting like this::
``
INSTALLED_APPS = [
...
'password_histories',
]
``

2. Migrate
``
python manage.py migrate
``
